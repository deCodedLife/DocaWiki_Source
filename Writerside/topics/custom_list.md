# Кастомный список

Кастомные списки необходимы для заранее определённых типов полей. Например: выбор пола, способа оплаты

## Структура

> \* Обязательные поля
{style="warning"}

| Параметр | Тип данных | Значение          |
|----------|:----------:|-------------------|
| * title  |   string   | Название значения |
| * value  |    any     | Значение          |


### Пример

```json
[
  {
    "title": "Мужской",
    "value": "M"
  },
  {
    "title": "Женский",
    "value": "W"
  }
]
```

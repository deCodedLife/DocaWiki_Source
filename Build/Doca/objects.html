Объекты

Объекты являются основополагающей структурой в проекте, с помощью которой описываются все сущности.

Структура

Описание объекта

* Обязательные параметры

Параметр

Тип данных

Описание

* title

string

Название объекта

* table

string

Название таблицы бд

* is_trash

bool

Хранить удалённую информацию

* properties

array of objects

Поля

action_buttons

array of objects

Операции над записями

Пример

{ "title": "Продажи", "table": "salesList", "is_trash": false, "properties": [], "action_buttons": [] }
